CyberGuard Assistant is a C# console-based chatbot designed to help users learn about cybersecurity and safe internet practices. The application interacts with users in a friendly conversational manner, providing responses to basic queries and promoting awareness of online safety.

 Features
 Interactive chatbot interface in the console
 Audio greeting (Windows-supported)
 Basic input validation
 Personalized conversation (asks for user's name)
 Predefined cybersecurity-related responses
 Styled console UI with headers and prompts# Prog-Part-1
How It Works
The program starts in Program.cs
A styled header is displayed using ConsoleUI
A greeting sound may play using AudioPlayer
The chatbot:
Asks for the user’s name
Continuously accepts user input
Validates input using InputValidator
Processes input using ResponseHandler
Displays responses in the console
